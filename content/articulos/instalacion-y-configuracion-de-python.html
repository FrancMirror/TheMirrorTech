¿Qué es python?

Python es un lenguaje de programación de alto nivel que se utiliza para desarrollar aplicaciones de todo tipo. A diferencia de por ejemplo Java, python se trata de un lenguaje interpretado, es decir, que no es necesario compilarlo para ejecutar las aplicaciones escritar en Python, sino que se ejecutan directamente en el ordenador utilizando un programa denominado interpretador, por lo que no es necesario "traducirlo a lenguaje máquina".

Este lenguaje es muy sencillo de leer y escribir debido a su alta similitud con el lenguaje humano. Además, se trata de un lenguaje multiplataforma de de código abierto y, por lo tanto, gratuito, lo que permite desarrollar software sin límites. Con el paso del tiempo, Python ha ido ganando adeptos gracias a su sencillez y a sus amplias posibilidades, sobre todo en los últimos años, ya que facilita trabajar con inteligencia artificial, big data, machine learning y data science, entre muchos otros campos que cada día están más en auge.

En este pdf podrás encontrar como poder instalarlo y configurarlo para que puedas empezar a trabajar con este lenguaje.
